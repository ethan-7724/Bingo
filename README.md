# Bingo

> A Personal AI Operating System.

Bingo 是一个拥有长期记忆、主动思考、任务管理和插件系统的个人 AI。

它不是一个聊天机器人，而是一个能够随着主人一起成长的 AI 系统。

---

## Vision

打造一个真正属于自己的 AI。

模型可以更换。

记忆不会丢失。

随着时间不断成长。

---

## Features（规划中）

- 🧠 Long-term Memory

- 🎮 Minecraft Agent
